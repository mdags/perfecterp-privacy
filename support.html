Uygulamayı İlk Kez Kurma

  1. Uygulamayı açın — ilk açılışta Kurulum Ekranı görüntülenir.
  2. API Adresi alanına ERP sunucu adresinizi girin.
    Örnek: http://192.168.1.100:7979/erpapi/api/
  3. Uygulama Adı alanına firmanızın adını yazın (isteğe bağlı).
  4. Muhasebe Programı seçimini yapın (Mikro / Netsis / Logo / Yok).
  5. Kaydet ve Devam Et butonuna tıklayın.
  6. Kullanıcı adı ve şifrenizle giriş yapın.
API adresini ERP sistem yöneticinizden alabilirsiniz. Adres genellikle şirket iç ağında ya da VPN üzerinden erişilebilen bir sunucu adresidir.

Sık Sorulan Sorular

Uygulamaya giriş yapamıyorum, ne yapmalıyım?

Önce aşağıdakileri kontrol edin:

  • Cihazınız ERP sunucusunun bulunduğu ağa bağlı mı? (Şirket Wi-Fi veya VPN)
  • API adresi doğru girilmiş mi? (Sondaki / işaretine dikkat edin)
  • Kullanıcı adı ve şifreniz doğru mu?
  • ERP sistem yöneticiniz hesabınızı aktif etmiş mi?

Sorun devam ederse destek ekibimizle iletişime geçin.

API adresini nasıl öğrenebilirim?

API adresini ERP sistem yöneticinizden alabilirsiniz. Genellikle şu formatta olur:

http://[SunucuIP]:7979/erpapi/api/

Şirket dışından erişmek istiyorsanız VPN bağlantısı gerekebilir.

Finans modülü görünmüyor, neden?

Finans modülü yalnızca kurulum ekranında bir muhasebe programı seçildiğinde aktif olur. Ayarlar dişlisine tıklayarak kurulum ekranını açın ve Mikro, Netsis veya Logo seçeneğini seçip kaydedin.

Veriler neden güncel görünmüyor?

Ekranı aşağı çekerek yenileme yapabilirsiniz (pull-to-refresh). Dashboard beş dakikada bir otomatik olarak yenilenir. Aktif duruş ekranı ise 60 saniyede bir otomatik güncellenir.

API adresini nasıl değiştirebilirim?

Giriş ekranının sağ üst köşesindeki dişli ikonuna tıklayın. Ayarlar ekranından API adresini, uygulama adını ve muhasebe programı seçimini güncelleyebilirsiniz.

Uygulama hangi ERP sistemleriyle çalışır?

Uygulama yalnızca Perfect ERP sistemiyle çalışır. Başka bir ERP sistemine bağlanmak mümkün değildir. Perfect ERP lisansı için satış ekibimizle iletişime geçebilirsiniz.

Ek rapor veya modül talep edebilir miyim?

Evet. İhtiyacınıza özel yeni raporlar veya modüller geliştirilebilir. Talep ve fiyat bilgisi için destek e-postamıza yazabilirsiniz.

Verilerimin güvenliği nasıl sağlanıyor?

Uygulama yalnızca kendi şirketinizin sunucusuyla iletişim kurar. Hiçbir veri üçüncü taraf sunuculara iletilmez. Detaylı bilgi için Gizlilik Politikamızı inceleyebilirsiniz.

Getting Started

  1. Open the app — the Setup Screen appears on first launch.
  2. Enter your ERP server API Address.
    Example: http://192.168.1.100:7979/erpapi/api/
  3. Enter your company name as the App Title (optional).
  4. Select your Accounting Software (Mikro / Netsis / Logo / None).
  5. Tap Save and Continue.
  6. Log in with your username and password.
Get the API address from your ERP system administrator. It is typically a server address accessible on the company's internal network or via VPN.

Frequently Asked Questions

I can't log in. What should I do?

Please check the following:

  • Is your device connected to the ERP server's network? (Company Wi-Fi or VPN)
  • Is the API address correct? (Make sure it ends with a /)
  • Are your username and password correct?
  • Has your ERP system administrator activated your account?

If the problem persists, contact our support team.

How do I find my API address?

Your API address can be obtained from your ERP system administrator. It typically follows this format:

http://[ServerIP]:7979/erpapi/api/

If you need remote access, a VPN connection may be required.

The Finance module is not visible. Why?

The Finance module is only active when an accounting software is selected during setup. Tap the settings gear icon on the login screen, open the Setup screen, and select Mikro, Netsis, or Logo, then save.

Why does the data look outdated?

Pull down on any screen to refresh manually. The Dashboard auto-refreshes every 5 minutes. The Active Downtime screen refreshes every 60 seconds.

How do I change the API address?

Tap the gear icon in the top-right corner of the login screen. From the Settings screen you can update the API address, app title, and accounting software selection.

Which ERP systems does this app work with?

This application works exclusively with Perfect ERP. Connecting to other ERP systems is not supported. Contact our sales team for Perfect ERP licensing information.

Can I request additional reports or modules?

Yes. Custom reports and additional modules can be developed on request. Please email our support team for details and pricing.

How is my data kept secure?

The app communicates only with your own company's server. No data is transmitted to third-party servers. For more information, please review our Privacy Policy.

Reporting a Bug

If you notice a bug or unexpected behavior, please send the following information by email:

  • Your device model and OS version
  • App version (visible in the bottom-left corner of the login screen)
  • When and on which screen the issue occurred
  • A screenshot if available
Please do not include sensitive information such as your username, password, or API address in your email.
