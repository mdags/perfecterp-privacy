Genel Bilgi

Perfect ERP Mobile, üretim işletmelerine özel bir kurumsal kaynak yönetimi (ERP) mobil raporlama uygulamasıdır. Uygulama yalnızca kurumsal (B2B) kullanıcılara lisanslanmakta olup genel kamuya yönelik değildir.

Bu gizlilik politikası; uygulamanın hangi verileri topladığını, bu verileri nasıl kullandığını ve koruduğunu açıklamaktadır.

Topladığımız Veriler

Uygulama yalnızca aşağıdaki bilgileri cihazda yerel olarak saklar:

  • Kullanıcının girdiği ERP sunucu API adresi
  • Uygulama görünen adı (isteğe bağlı)
  • Muhasebe programı seçimi (Mikro, Netsis, Logo veya Yok)
  • Oturum kimlik doğrulama tokeni (Bearer Token)
Bu bilgilerin hiçbiri bizimle veya herhangi bir üçüncü tarafla paylaşılmaz. Tüm veriler yalnızca kullanıcının kendi cihazında (SharedPreferences) saklanır.

Toplamadığımız Veriler

Uygulama aşağıdaki bilgileri kesinlikle toplamaz:

  • Ad, soyad, e-posta adresi veya telefon numarası
  • Konum bilgisi (GPS veya ağ tabanlı)
  • Kamera veya mikrofon verisi
  • Kişi rehberi
  • Cihaz kimliği veya reklam kimliği (IDFA / GAID)
  • Kullanım istatistikleri veya analitik veri
  • Çerez (cookie) veya izleme verisi

Veri İletişimi

Uygulama yalnızca kullanıcının kendi bünyesinde çalışan özel ERP sunucusuyla iletişim kurar. Bu sunucu kamuya açık internet üzerinde değil; müşteri firmanın yerel ağında ya da VPN üzerinden erişilebilen özel bir ortamda çalışmaktadır.

Uygulama herhangi bir reklam ağına, analitik servisine veya üçüncü taraf bulut sistemine veri iletmez.

Talep Edilen İzinler

Uygulama yalnızca şu izni talep eder:

  • İnternet erişimi — ERP sunucusuna bağlanmak için zorunludur.

Kamera, konum, mikrofon veya kişi rehberi erişimi talep edilmemektedir.

Çocukların Gizliliği

Bu uygulama yalnızca kurumsal kullanıcılara yöneliktir ve 18 yaşın altındaki bireylere hitap etmemektedir. Bilerek hiçbir çocuktan kişisel veri toplamıyoruz.

Overview

Perfect ERP Mobile is a corporate mobile reporting application for manufacturing businesses. It is licensed exclusively on a B2B basis to enterprise clients and is not intended for the general public.

This Privacy Policy explains what data the application collects, how it is used, and how it is protected.

Data We Store

The application stores only the following information locally on the device:

  • ERP server API address entered by the user
  • Application display name (optional)
  • Accounting software selection (Mikro, Netsis, Logo, or None)
  • Session authentication token (Bearer Token)
None of this information is shared with us or any third party. All data is stored exclusively on the user's own device (SharedPreferences).

Data We Do NOT Collect

The application does not collect any of the following:

  • Name, email address, or phone number
  • Location data (GPS or network-based)
  • Camera or microphone data
  • Contacts or address book
  • Device identifiers or advertising IDs (IDFA / GAID)
  • Usage statistics or analytics data
  • Cookies or tracking data of any kind

Data Transmission

The application communicates exclusively with the user's own private ERP server. This server is not accessible on the public internet — it operates within the client company's private local network or through a VPN.

The application does not transmit any data to advertising networks, analytics services, or any third-party cloud systems.

Permissions Requested

The application requests only the following permission:

  • Internet access — required to connect to the ERP server.

No access to camera, location, microphone, or contacts is requested.

Children's Privacy

This application is intended exclusively for enterprise users and is not directed at individuals under the age of 18. We do not knowingly collect personal data from children.
